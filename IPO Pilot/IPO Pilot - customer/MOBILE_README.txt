# IPO~Master Mobile Binary
# This is a Linux ARM64 binary compiled for Android/Termux
# 
# To use on Android:
# 1. Install Termux (from F-Droid preferred)
# 2. Transfer this file to your phone
# 3. In Termux: chmod +x ipo_master_mobile && ./ipo_master_mobile
#
# Requirements: Android device with Termux installed
# Architecture: ARM64 (works on most modern Android devices)
